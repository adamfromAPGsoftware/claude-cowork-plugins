# Creative Data Schema

## Location

`marketing-plugin/data/creative-data.json` — registry of all generated ad creatives, organized by batch.

## Top-Level Structure

```json
{
  "last_generation": "ISO 8601 timestamp of last creative generation",
  "batches": [ ... ]
}
```

| Field | Required | Description |
|-------|----------|-------------|
| last_generation | Yes | Timestamp of most recent generation run (null if never generated) |
| batches | Yes | Array of creative batches, each containing angles and generated assets |

## Batch

```json
{
  "batch_id": "string — YYYYMMDD-HHMMSS format",
  "created_at": "ISO 8601",
  "campaign_id": "string | null — links to campaign-data.json campaign_id",
  "source": {
    "competitor_winners": ["ad_id", "..."],
    "market_intelligence_report": "string — file path or null",
    "performance_data": "string — summary of own campaign performance used"
  },
  "angles": [ ... ]
}
```

| Field | Required | Description |
|-------|----------|-------------|
| batch_id | Yes | Unique batch identifier, used in file paths. **Dedup key.** |
| created_at | Yes | When this batch was created |
| campaign_id | No | Links this batch to a campaign in campaign-data.json. Null for standalone batches. |
| source | Yes | What inputs were used to generate this batch |
| source.competitor_winners | No | Ad IDs from competitor-data.json that inspired this batch |
| source.market_intelligence_report | No | Path to the market intelligence report used |
| source.performance_data | No | Summary of own campaign metrics that informed angle selection |
| angles | Yes | Array of creative angles within this batch |

## Angle

```json
{
  "angle_id": "string — batch_id-NNN sequence",
  "angle_name": "string — short descriptive name",
  "hook_line": "string — the opening hook",
  "copy_variants": [ ... ],
  "images": [ ... ],
  "videos": [ ... ],
  "landing_page_url": "string — target URL for these ads",
  "framework": "PAS | AIDA | BAB | Curiosity Gap | Social Proof",
  "source": "string — what insight drove this angle",
  "ad_scripts": [ ... ],
  "status": "draft | images_generated | videos_generated | scripts_generated | packaged | retired",
  "meta_ad_ids": ["string — Meta ad IDs after upload"],
  "uploaded_at": "ISO 8601 | null",
  "retired_at": "ISO 8601 | null",
  "retirement_reason": "string | null — e.g. 'CPA $45 exceeded max $30 after 14 days'",
  "performance_snapshot": {
    "date_range": "string — YYYY-MM-DD to YYYY-MM-DD",
    "ad_count": "number — how many ads were linked",
    "spend": "number",
    "impressions": "number",
    "clicks": "number",
    "ctr": "number — percentage",
    "cpc": "number",
    "conversions": "number",
    "cpa": "number",
    "roas": "number",
    "verdict": "winner | loser | okay | insufficient_data",
    "snapshot_date": "ISO 8601 — when this snapshot was taken"
  }
}
```

| Field | Required | Description |
|-------|----------|-------------|
| angle_id | Yes | Unique within batch. Format: `{batch_id}-001`, `{batch_id}-002`, etc. |
| angle_name | Yes | Human-readable angle description |
| hook_line | Yes | The opening hook/attention grabber |
| framework | No | Which angle framework was used (PAS, AIDA, BAB, Curiosity Gap, Social Proof) |
| source | No | What insight drove this angle (competitor pattern, market theme, gap) |
| inspiration_ads | Yes | Array of 1-3 specific competitor ads that inspired this angle's design and messaging |
| copy_variants | Yes | Array of copy variants for this angle |
| images | No | Array of generated image creatives |
| videos | No | Array of generated video creatives |
| ad_scripts | No | Array of filmable ad video scripts generated by [GS] |
| landing_page_url | No | Where ads for this angle should send traffic |
| status | Yes | Current state of this angle's creative production (includes `retired` for stopped angles) |
| retired_at | No | When this angle was retired due to poor performance |
| retirement_reason | No | Why this angle was retired (metric + threshold that triggered it) |
| meta_ad_ids | No | Array of Meta ad IDs after the creative was uploaded to Ads Manager |
| uploaded_at | No | When the creative was uploaded to Meta |
| performance_snapshot | No | Aggregated performance data from linked Meta ads (populated by LP capability) |

## Inspiration Ad

```json
{
  "ad_id": "string — competitor ad ID from competitor-data.json",
  "page_name": "string — competitor page name",
  "local_path": "string — path to downloaded creative asset",
  "why": "string — what design/angle elements are being adapted from this winner"
}
```

| Field | Required | Description |
|-------|----------|-------------|
| ad_id | Yes | The competitor ad ID, must exist in competitor-data.json |
| page_name | Yes | Competitor page name for quick reference |
| local_path | Yes | Path to the downloaded competitor creative image/video |
| why | Yes | What specific design elements, layout patterns, or angle themes are being adapted |

Every angle MUST have 1-3 inspiration ads. These serve two purposes: (1) traceability — know which winners inspired which creatives, and (2) style reference — the local_path images are passed to the image generation script via `--inspo-dir` so the model can see and adapt the winning design.

## Copy Variant

```json
{
  "variant": "direct | story | question",
  "primary_text": "string — main ad body text",
  "headline": "string — ad headline",
  "description": "string — link description",
  "cta": "LEARN_MORE | SIGN_UP | SHOP_NOW | BOOK_NOW | GET_OFFER | CONTACT_US"
}
```

| Field | Required | Description |
|-------|----------|-------------|
| variant | Yes | Copy style: `direct` (straightforward), `story` (narrative), `question` (curiosity-driven) |
| primary_text | Yes | The main ad body text (Facebook primary text field) |
| headline | Yes | Short headline shown below the creative |
| description | No | Link description text |
| cta | Yes | Call-to-action button type |

## Image Creative

```json
{
  "aspect": "1:1 | 9:16 | 1.91:1",
  "path": "string — relative path to generated file",
  "generated_at": "ISO 8601",
  "model": "nano-banana-pro",
  "prompt_used": "string — the generation prompt"
}
```

| Field | Required | Description |
|-------|----------|-------------|
| aspect | Yes | Aspect ratio matching Meta ad format: `1:1` (feed square), `9:16` (stories/reels), `1.91:1` (feed landscape) |
| path | Yes | Relative path from plugin root to the generated image file |
| generated_at | Yes | When the image was generated |
| model | Yes | Model used for generation |
| prompt_used | Yes | The exact prompt sent to the model (for reproducibility) |

## Video Creative

```json
{
  "aspect": "16:9 | 9:16 | 1:1",
  "path": "string — relative path to generated file",
  "model": "kling | veo",
  "duration_seconds": 5,
  "generated_at": "ISO 8601",
  "prompt_used": "string — the generation prompt",
  "reference_image": "string | null — path to input image for image-to-video"
}
```

| Field | Required | Description |
|-------|----------|-------------|
| aspect | Yes | Video aspect ratio |
| path | Yes | Relative path to the generated video file |
| model | Yes | `kling` (Kling 3.0 via fal.ai) or `veo` (Veo 3.1 via fal.ai) |
| duration_seconds | Yes | Video duration (typically 5 or 10 seconds) |
| generated_at | Yes | When the video was generated |
| prompt_used | Yes | The exact prompt sent to the model |
| reference_image | No | Path to input image (for Kling image-to-video mode) |

## Ad Script

```json
{
  "script_id": "string — {angle_id}-gs-{NN}",
  "path": "string — relative path to script markdown file",
  "duration": "number — target duration in seconds",
  "word_count": "number — total word count across all sections",
  "cta_keyword": "string — AUDIT | WASTE | AI | RESULTS | none",
  "status": "script_written | filmed | edited | uploaded",
  "filmed_video_path": "string | null — path to raw filmed landscape video",
  "edited_video_path": "string | null — path to final 9:16 vertical after SF pipeline"
}
```

| Field | Required | Description |
|-------|----------|-------------|
| script_id | Yes | Unique script identifier: `{angle_id}-gs-01` |
| path | Yes | Relative path to the script markdown file |
| duration | Yes | Target duration in seconds (15-30s) |
| word_count | Yes | Total words across hook + body + CTA |
| cta_keyword | Yes | Comment-gated keyword (`AUDIT`, `WASTE`, `AI`, `RESULTS`) or `none` |
| status | Yes | Lifecycle state: `script_written` → `filmed` → `edited` → `uploaded` |
| filmed_video_path | No | Path to raw landscape video after Adam films it |
| edited_video_path | No | Path to final 9:16 vertical after Video Editor SF pipeline processing |

## File Storage

```
data/creatives/
└── {batch_id}/
    ├── UPLOAD-GUIDE.md          — Meta Ads Manager upload instructions
    ├── {angle_id}-1x1.png       — Square feed image
    ├── {angle_id}-9x16.png      — Stories/Reels image
    ├── {angle_id}-191x1.png     — Landscape feed image
    ├── {angle_id}-9x16.mp4      — Vertical video (AI-generated)
    ├── {angle_id}-1x1.mp4       — Square video (AI-generated)
    └── scripts/
        ├── {angle_id}-gs-01.md  — Filmable ad script
        ├── gs-all-teleprompter.md — Combined teleprompter for filming session
        └── raw/                 — Raw filmed landscape videos
```

## Status Progression

```
draft → images_generated → videos_generated → scripts_generated → packaged → retired (terminal)
```

- `draft` — angles and copy created, no assets yet
- `images_generated` — image creatives produced for all required formats
- `videos_generated` — video creatives also produced (AI-generated via Veo/Kling)
- `scripts_generated` — filmable ad scripts written by [GS] (may coexist with images/videos status)
- `packaged` — UPLOAD-GUIDE.md generated, ready for manual Meta upload
- `retired` — angle retired due to poor performance (set by Performance Review)
