# Generate Edit Instructions

Produce a self-contained edit instructions file that the video editor (vid-1 [VE]) can consume directly. This file contains everything the editor needs: section map, MG insertion points, emphasis text cues, pacing rules, and asset checklist.

## Process

1. **Load the generated VSL script** from the current project in vsl-data.json.

2. **Load VSL pacing rules** — Read `{project-root}/video-plugin/skills/1-video-editor/workflows/vsl-edit/data/vsl-pacing-rules.md` (V1–V12).

3. **Load VSL compliance checklist** — Read `{project-root}/video-plugin/skills/1-video-editor/workflows/vsl-edit/data/vsl-compliance-checklist.md`.

4. **Generate edit instructions file** at `{project-root}/content/standalone/{date}-{slug}/vsl-edit-instructions.md`:

   ```markdown
   # VSL Edit Instructions — {offer name}

   Generated by VSL Generator [GE]. Load this file when invoking vid-1 [VE].

   ---

   ## Section A: Edit Profile

   This is a VSL, not a YouTube tutorial. Apply these overrides to the standard editing pipeline:

   - **Structure:** Single continuous video — NO intro/body split
   - **Pacing:** 20-25 layout changes per minute throughout (V1)
   - **Speaker max hold:** 8 seconds without an overlay (V2)
   - **Zoom:** NO jump-cut zooms — speaker framing stays static (T2)
   - **Transitions:** 100% hard cuts — zero dissolves or fades (T1)
   - **Audio:** Single continuous voiceover — never breaks at visual transitions (T3, V12)

   ---

   ## Section B: Script Section Map

   | # | Section | Estimated Time | Duration | Format | Speaker % | Graphic % |
   |---|---------|---------------|----------|--------|-----------|-----------|
   {table from script sections}

   ---

   ## Section C: Motion Graphic Insertions

   Map each graphic to its approximate insertion point in the recorded video. Align to transcript using the section boundary text cues.

   | # | Section | Trigger Text | MG Name | Duration | MG File | Type |
   |---|---------|-------------|---------|----------|---------|------|
   {table from script graphics}

   **Alignment method:** Use word-level transcript timestamps to find the trigger text. Insert the graphic starting at that timestamp. The trigger text is the last spoken phrase BEFORE the graphic should appear.

   ---

   ## Section D: Emphasis Text Cues

   Large text breakouts that should appear beside the speaker at key moments.

   | # | Trigger Text | Display Text | Highlight Words (Green) | Duration |
   |---|-------------|--------------|------------------------|----------|
   {table of emphasis text moments from script — thesis statements, key phrases}

   ---

   ## Section E: Icon Pop-Up Cues

   Tool/brand logos that should spring-animate when mentioned.

   | # | Trigger Text | Icon | Position |
   |---|-------------|------|----------|
   {table of tool mentions from script}

   ---

   ## Section F: B-Roll Cues

   B-roll segments with B&W treatment and aggressive text overlay.

   | # | Section | Trigger Text | B-Roll Description | Overlay Text | Duration |
   |---|---------|-------------|-------------------|--------------|----------|
   {table of B-roll moments from script}

   ---

   ## Section G: Subtitle Highlights

   Words that should be highlighted in green (#72E032) in the subtitles. These are the **bold** words from the script.

   {list of emphasis words per section}

   ---

   ## Section H: Asset Checklist

   All assets needed for this VSL edit:

   ### Motion Graphics (Hera API)
   | # | Name | Duration | Hera Prompt | Status |
   |---|------|----------|-------------|--------|
   {table with full Hera prompts from script}

   ### B-Roll Footage
   | # | Description | Treatment | Source |
   |---|-------------|-----------|--------|
   {table of B-roll needed}

   ### Icon Assets
   | # | Tool/Brand | Logo File |
   |---|-----------|-----------|
   {table of icons needed}

   ---

   ## Section I: Quality Targets

   - **Runtime target:** ~{minutes} at natural speaking pace
   - **Visual split:** {speaker}% speaker / {graphic}% graphics / {broll}% B-roll
   - **Layout changes/min:** 20-25 (V1)
   - **Total full-screen MGs:** {count}
   - **Total emphasis text breakouts:** {count}
   - **Subtitle coverage:** 100% of speaker dialogue
   ```

5. **Update vsl-data.json** — Set `edit_instructions_path`, `status` to `ready_for_edit`.

6. **Present handoff summary:**

   ```
   Edit instructions saved to: {path}

   To produce the video:
   1. Record the VSL using the teleprompter script at {script_path}
   2. Drop the recording into {project_folder}/video-editor/video-ingest/
   3. Place the edit instructions file alongside it (or it will auto-load from project folder)
   4. Invoke vid-1 [VE] — it will load the edit instructions and apply VSL-specific editing

   Assets to prepare before editing:
   - {count} Hera motion graphics (generate with vid-1 [HM])
   - {count} B-roll clips ({source description})
   - {count} icon/logo assets

   Ready to record!
   ```
