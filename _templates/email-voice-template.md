# Email Voice Reference

Load this before generating ANY client-facing email or document. These rules are non-negotiable.

---

## Greeting

- **"Hey {Name},"** — default for most contacts (post-discovery, ongoing relationship)
- **"Hi {Name},"** — first email to corporate/professional-services contacts only
- **NEVER:** "Dear {Name}," — too formal

## Opening

- Get to the point. First sentence references something specific — a number they said, a process they described, a pain they expressed.
- No "I hope this email finds you well." No "It was great to chat." No pleasantries.
- Acknowledges then acts: "Thanks for your patience. I was able to take a deeper look..."

## Closing

- **"Thanks,"** — most common professional sign-off
- **"Talk soon,"** — warm, post-call
- **"Let me know what you think!"** — when seeking input
- **"Cheers,"** — casual
- Always followed by just **"{YOUR_NAME}"** on the next line, then **"{YOUR_COMPANY}"** below that.
- **NEVER:** "Best regards", "Kind regards", "Warm regards", "Yours sincerely", "I hope this helps"

## Structure

- 1-3 paragraphs max. Short paragraphs (1-3 sentences, sometimes just one).
- No preamble, no summary at the end.
- Vary paragraph length — a 1-sentence paragraph followed by a longer one is natural.
- Max 1 exclamation mark per email, on something genuinely positive.
- No bold text for emphasis in body copy.
- No bullet lists unless genuinely listing multiple items (used sparingly).
- Comma-separated flowing thoughts rather than over-structured lists.

## Grammar

- **Contractions mandatory:** can't, don't, it's, you're, I've, we're, that's, there's, isn't, hasn't.
- **Numbers as numerals always:** "4-6 weeks", "$2,500", "3 staff"
- **Start sentences with:** "And", "But", "So" — conversational tone.
- **Em dashes sparingly** — max 1-2 per email for genuine pauses.
- **Sentence case** for any headings — never Title Case.

## Natural Qualifiers

Use these to sound human. At least one per email:
- **"just"** — "just thought I'd let you know", "just to confirm"
- **"obviously"** — natural qualifier, not condescending
- **"probably"** — soft opinions: "This would probably be the best approach"
- **"I think..."** — tentative opinions, not hedged expertise
- **"So essentially..."** — signals a summary

## Forbidden Phrases — Never Use

| AI/Corporate Pattern | Better Version |
|---------------------|----------------|
| "In conclusion" | Just end. Or: "So here's where that leaves us —" |
| "It's important to note that" | Drop it. Just say the thing. |
| "In today's digital landscape" | Delete entirely. |
| "Delve into" | "jump into" / "dig into" |
| "Leverage" / "Utilize" | "use" |
| "Facilitate" | "help" / "make it easier" |
| "Furthermore" / "Additionally" / "Moreover" | "And" / "Now" / just start the next sentence |
| "It is important to" / "It is essential to" | "You need to" / just state it |
| "I hope this helps" | Never. |
| "Please don't hesitate to reach out" | Never. |
| "Feel free to reach out" | Never. |
| "I hope this email finds you well" | Never. Start with the actual thing. |
| "Going forward" | "from now on" |
| "Seamless" / "Innovative" / "Comprehensive" | Specific description of what it does |
| "Elevate" / "Groundbreaking" / "Game-changing" | Specific impact |
| "Synergy" / "Ecosystem" | Specific description |
| "As an AI..." | Never. |
| "Please find attached" | "Attaching X here." / just attach it |

## Structural Anti-Patterns — Avoid

- **Perfectly parallel bullet lists** (same length) — mix fragments with full sentences
- **Bold keywords every paragraph** — bold nothing, or one critical thing per section max
- **Every paragraph the same length** — vary: 1-sentence then longer
- **Predictable transition words** opening every paragraph — vary or drop them
- **Three-sentence intro before the point** — start with the point

## Industry Tone Calibration

| Industry Tag | Tone |
|-------------|------|
| Home services / construction / trades | Plain, direct, no jargon |
| Healthcare / support work | Warm but still specific and fast |
| Real estate / property | Professional, numbers-forward |
| Corporate / professional services | Crisp, ROI-framed |

## Voice-Check Gate

Before presenting ANY email or client-facing output, ask yourself:

> **"Would {YOUR_NAME} say this in a casual conversation?"**

If no — rewrite it. Then scan against the forbidden phrases table.
