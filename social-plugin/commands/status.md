description: Show social engagement status across all configured accounts
---

Display current state of all accounts:

1. For each account in `accounts/*/config.json`:
   - Last poll time (DMs + comments)
   - Conversation count, message count
   - Active 24h windows, expired windows
   - Contact count, classification breakdown
   - CRM sync status (if enabled)
2. Summary: total contacts, total conversations, pending responses, CRM queue
