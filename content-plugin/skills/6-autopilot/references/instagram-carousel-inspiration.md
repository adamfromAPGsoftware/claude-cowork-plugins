## Pattern Analysis

### What makes these carousels outliers

**1. Specificity is the multiplier**
- "4 prompts that run your business" → 14K comments
- "50 skills you can use" → 1.4K comments
- Exact number + exact outcome > big claim

**2. The comment-CTA mechanic is a lead gen engine**
- Every post uses a single keyword CTA (CLAUDE, PROMPTS, SKILLS, ESSENTIALS)
- Creator responds personally to every comment — appears in all reply threads
- The comment itself is the DM trigger (ManyChat automation implied)
- Comments spike engagement signals → algorithm distributes further → more comments

**3. Verbatim prompts as slide content**
- Audience can screenshot the prompt and use it immediately
- No gatekeeping until slide 6+ (CTA) — full value delivered in the carousel itself
- Creates trust: "this actually works, I can try it right now"

**4. Self-referential proof**
- "This post was made with the IG Carousel Generator skill" (Post 4, Slide 3)
- Demonstrates the tool while showing its output
- Strongest credibility mechanic in the set

**5. Caption structure that works**
- Bold claim first (no soft opener)
- "Most people think X. It's not." — reframes the audience's assumption
- Numbered list in caption mirrors the numbered slides
- Final line = single keyword CTA only

### Design patterns

- **Aspect ratio:** 1080×1254 (4:3.5) or 1080×1350 (4:5) — all portrait
- **Background:** Near-black (#0a0a0a to #1a1a1a range)
- **Layout:** Magazine/crossword overlay style — dense text blocks with clear hierarchy
- **Typography:** Bold white headline, smaller body text, monospace for prompts/code
- **Per-slide structure:** Large category label (top) → title → content block → @handle + save CTA (bottom)
- **Save CTA:** "save" / "save for later" watermark appears on almost every slide
- **No hashtags in caption** (Post 1, 2, 3) vs. hashtags (Post 4, 5) — @itstylergermain skips hashtags entirely

### Caption voice markers (@itstylergermain)

- All lowercase opener ("i built", "most people")
- No hashtags on top-performing posts
- Short punchy lines — no paragraph blocks
- Conviction statements ("none of these are hypothetical, I use all of them")
- Anti-positioning ("most people treat X like Y. It's not")

### What to steal for creator carousels

1. **Format:** Verbatim Claude Code / prompt text as slide content — audience gets immediate value
2. **Hook structure:** "X thing I [do/built/use] every day" or "If I had to [constraint], here's what I'd do"
3. **CTA mechanic:** Single keyword → ManyChat DM (already have this infrastructure)
4. **Caption opening:** Lowercase, bold claim, reframe assumption in line 2
5. **Slide count:** 6-8 is the sweet spot — enough to deliver value, short enough to hold attention
6. **Self-referential proof:** When the carousel topic IS the tool, say so on a slide
