# IC Topic Angle Rotation

The IC workflow rotates through 5 topic angles to keep carousel content varied.
Each angle has dedicated Exa search queries and preferred CTA keywords.

## Rotation Sequence

```
[claude-updates, ai-tools, ai-business, ai-agents, community]
```

---

## Angle Definitions

### 1. claude-updates
**Focus:** Latest Anthropic/Claude product news — new features, model releases, capability improvements
**Exa queries:**
- "Claude Code new feature 2026" (last 48h)
- "Anthropic announcement release" (last 48h)
- "Claude model update capability" (last 48h)
**Preferred CTA keywords:** CLAUDE, UPDATE, BUILD
**Example carousel concepts:**
- "5 things Claude can do now that it couldn't 3 months ago"
- "Claude just shipped [feature] — here's what it means for AI builders"
- "The Claude Code update everyone is sleeping on"
