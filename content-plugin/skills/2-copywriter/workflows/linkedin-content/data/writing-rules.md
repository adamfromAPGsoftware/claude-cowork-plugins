# LinkedIn Writing Rules

## Universal Rules (All Formats)

- Short sentences. Punchy delivery. No corporate speak
- Each line must earn the next line — if it doesn't add value, cut it
- Use line breaks liberally — LinkedIn mobile favours vertical white space
- Bold or CAPS sparingly for emphasis (1-2 per post max)
- ZERO hashtags — never include hashtags anywhere in the post copy, comments, or any output. Hashtags are an immediate AI-generated tell and must never appear
- Dual-funnel check: builders see a skill, SMEs see ROI
- Reference inspiration library hook patterns and proven post structures
- Source from categorised inspiration library before writing — study real examples

## Format-Specific Rules

### Text Post (800-1300 characters)

- Pure copy must carry the weight — no visual support
- Use → arrow formatting for lists (signature visual rhythm)
- Credentials woven mid-post, not upfront — supporting evidence, never opening brag
- Include an honest P.S. line where appropriate (builds trust, counters LinkedIn hype)
- Highest emphasis on hook quality — text-only lives or dies on the open

### Image Post (800-1300 characters)

- Post text should reference what the viewer will see — don't write generic text that ignores the visual
- Image and hook should amplify each other
- Each paragraph 1-2 sentences max for mobile readability
- Line breaks between every paragraph

### Carousel Post (600-1000 characters for post text)

- Post text accompanies the carousel PDF — it should tease, not duplicate
- Hook with the same angle as carousel title slide but different wording
- Tease the carousel content ("Swipe through to see...", "I broke down the 5 reasons...")
- Post text can be shorter since the carousel carries the value
- CTA keyword in post text must match the CTA slide's button text

#### Carousel Slide Rules

- Each headline is exactly 3 short lines (2-4 words per line)
- One line is the "highlight" (rendered in green)
- Body text is 1-2 sentences max per slide — punchy, not paragraphs
- Title slide: bold hook headline + supporting body + "SWIPE >>>" prompt
- Body slides: each makes ONE point, numbered automatically
- CTA slide: headline question + CTA button text + supporting body
- Total slides: 5-12 (sweet spot is 6-8 for engagement)
- Body slides build a logical sequence (problem → insight → proof → solution)

### Video Post (800-1300 characters)

- Post text should reference what the viewer will see — don't write generic text
- Promise something visual — a demo, a result, a process
- Video posts are project-based only — require existing video assets
- Video clips typically 15-30s, often sped up

## Post Body Structure (All Formats)

1. **Hook (line 1-2):** Stop the scroll in under 2 seconds
2. **Value section (3-8 lines):** Core insight, ROI, or skill
3. **Proof/specificity (2-3 lines):** Concrete details, numbers, outcomes
4. **CTA (final 1-3 lines):** Style A or Style B (see cta-patterns.md)

## Post Structure Templates

> Named templates derived from 5 proven post formats and top-performing external examples. Reference these when structuring post body — pick the template that matches the content category and angle.

### 1. Core Template (Lead Magnet / CTA)
Best for: Lead magnet posts with comment keyword CTA. The creator's highest-performing structure.
```
[Contrarian/alarming hook — 1 line]

[3-way problem framing OR highs vs reality contrast]

[Content teaser — "I just filmed/recorded/built..."]

[Value list with → arrows or ✅ checkmarks]

[Reframe line — what this is really about]

[Exclusivity/urgency — "early access" / "before it goes live"]

Comment "[KEYWORD]" and I'll send you [specific deliverable].

P.S. — [Honest caveat or practical benefit]
```

### 2. Vulnerability Story Template
Best for: Personal/authentic posts. No hard CTA — engagement from resonance.
```
[Statement that challenges belonging/identity]

[2-3 paragraphs of honest personal narrative]

[Turning point — what changed]

[Call to community — "If you've experienced this too..."]
```

### 3. Contrarian Anti-List Template
Best for: Personal posts that challenge conventional wisdom. High shareability.
```
[Bold claim about what I achieved]

Here's what I DIDN'T do:
❌ [Common advice 1]
❌ [Common advice 2]
❌ [Common advice 3]

All I did was [simple truth].

[Reframe to core principle]
```

### 4. Nurture / Educational Template
Best for: Value/educational posts with soft CTA. YouTube trailer companion posts.
```
[Outcome hook — what the result was]

[Context — why this matters / what prompted it]

[Method breakdown with → arrows or numbered steps]

[Key takeaway — single quotable sentence]

[Soft CTA — "Link in comments" / "Resource attached"]
```

## Performance Data by Category

> Enriched stats from 48-post analysed dataset + 5 proven post formats. Use to calibrate expectations and format choices.

### Lead Magnet Posts
- **Average engagement:** 1,184 (highest of all categories)
- **Comment:like ratio:** 1.8:1 to 2.4:1 (comment-heavy — CTA mechanic drives volume)
- **Best format:** Image (7/9 top posts use image format)
- **Average shares:** 1.6/post (lowest — lead magnets are consumed selfishly)
- **Key insight:** 2-step CTA (like + comment keyword) with ALL-CAPS keyword drives highest comment volume

### Personal / Authentic Posts
- **Average engagement:** 277
- **Like:comment ratio:** 7.4:1 (silent agreement pattern — high likes, fewer comments)
- **Best format:** Image (8/11 posts)
- **Average shares:** 6.5/post (mid-range — shares correlate with emotional relief)
- **Key insight:** Vulnerability and balanced takes outperform hot takes. No keyword CTA — open questions only

### Nurture / Educational Posts
- **Average engagement:** 347
- **Like:comment ratio:** 8.5:1 (passive consumption — readers learn, don't debate)
- **Best format:** linkedinVideo (avg 406 eng) > document (avg 323) > image (avg 174)
- **Average shares:** 16.5/post (highest — most redistributed category)
- **Key insight:** Named frameworks (WAT, DOE) and analogies drive highest shares. Soft CTA only

### Format Performance Comparison
- Carousels outperform text posts 1.6x for reach (6.60% avg engagement rate)
- linkedinVideo averages 406 engagement in nurture (algorithmic preference + demo proof)
- Document format with PDF attachment drives saves and bookmarks
- Text-only works when copy quality is elite (all 5 proven post formats are text-only)
- Comment keyword CTAs proven to drive 80-780+ comments on lead magnet posts
- Text-only comment keyword CTAs driving highest comment:like ratio at 4.9:1
