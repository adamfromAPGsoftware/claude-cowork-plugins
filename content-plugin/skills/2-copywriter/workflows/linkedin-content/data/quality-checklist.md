# Quality Checklist

## Universal Checks (All Formats)

- [ ] Hook stops the scroll (specific, curiosity-driven, not generic)
- [ ] Value delivered without needing to click anything
- [ ] Dual-funnel: builders see skill, SMEs see ROI
- [ ] CTA triggers comment with specific keyword (if lead magnet category)
- [ ] Mobile-friendly formatting (short paragraphs, line breaks)
- [ ] No angle duplication with other posts or channels
- [ ] ZERO hashtags anywhere — no hashtags in post body, comments, or any output (immediate AI tell)
- [ ] Post references proven structure template (see writing-rules.md Post Structure Templates)
- [ ] 3-category balance considered (lead magnet / personal / nurture)

## Text Post Checks

- [ ] Post length 800-1300 characters
- [ ] → arrow formatting used for lists (brand-consistent)
- [ ] Credentials woven mid-post, not upfront
- [ ] Pure copy carries the weight without visual support

## Image Post Checks

- [ ] Post length 800-1300 characters
- [ ] Image supports the hook (not generic stock-style)
- [ ] Image uses branded {YOUR_COMPANY} template where applicable
- [ ] Post text references what the viewer sees in the image

## Carousel Post Checks

- [ ] Post text length 600-1000 characters
- [ ] Carousel slides follow narrative arc (hook → build → CTA)
- [ ] Each slide headline is exactly 3 lines with one highlighted
- [ ] Carousel uses branded {YOUR_COMPANY} template
- [ ] Total slides: 5-12 (sweet spot 6-8)
- [ ] CTA slide keyword matches post text keyword
- [ ] Slides JSON config is valid and complete

## Video Post Checks

- [ ] Post length 800-1300 characters
- [ ] Video instructions clear enough to act on (clip path, speed, duration, trim points)
- [ ] Video shows something visually compelling (demo, result, process)
- [ ] Post text references what the viewer will see
