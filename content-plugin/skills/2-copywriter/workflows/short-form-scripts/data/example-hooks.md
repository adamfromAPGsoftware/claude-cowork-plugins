# Example Hooks — Short-Form Script Style References

6 annotated real hooks from top-performing inspiration videos. Use these as style models when writing hooks. Match on `angle_type` + `hook_pattern` to find the most relevant example for each script.

**Note:** All 6 examples are Primary ICP (TOFU). Secondary ICP voice has no direct examples in the dataset — use the Language Register Guide in `icp-profiles.md` instead.

---

### Example 1: Step-by-Step Tease (Automation Journey)

**Hook line:** "Here's how to make YouTube Shorts that post themselves, step by step."
**Source:** @nateherkai | 252K likes / 2.82M plays
**Angle type:** Story
**ICP:** Primary (TOFU)
**Hook pattern:** Step-by-Step Tease
**Emotional register:** Curious / Energised

**Why it works:**
- Promises a complete system ("post themselves") not just a tip — signals high value, low effort for viewer
- "Step by step" is the single highest-engagement hook phrase in the dataset (only 1/15 use it but it hit highest individual engagement) — it promises a full journey, not a teaser

**Body structure:** Mini-journey through each automation component in chronological build order
**CTA pattern:** Comment-gated → template or guide resource
**Visual style:** Coding/automation interface in top half of split-screen, fast cursor movement

**Adapt like this:**
"Here's how to [make/build/deploy] [specific outcome] that [desirable automated behaviour], step by step."
Examples:
- "Here's how to build client onboarding that runs itself, step by step."
- "Here's how to set up AI research that posts itself, step by step."

### Example 2: Value-Drop (Free Resource Reveal)

**Hook line:** "Christmas just came early for anyone trying to learn AI..."
**Source:** @nateherkai | 113K likes / 744K plays
**Angle type:** Value-Drop
**ICP:** Primary (TOFU)
**Hook pattern:** Value-Drop
**Emotional register:** Relieved / Grateful

**Why it works:**
- Seasonal/event framing creates an emotional trigger ("Christmas came early") that feels personal and lucky — viewer feels they're getting something others are missing
- Targets the identity ("anyone trying to learn AI") without being exclusionary — broad but specific enough to self-qualify

**Body structure:** Reveal the free resource, show why it's worth it, demonstrate one use case
**CTA pattern:** Comment-gated → access to the free resource
**Visual style:** Tool logo / resource preview in split-screen top half

**Adapt like this:**
"[Positive event framing] for anyone [doing/trying to/who wants to] [ICP goal]..."
Examples:
- "Big news for anyone trying to land their first AI client..."
- "This just dropped for anyone building with Claude Code..."

### Example 3: Product Drop (New Tool Launch)

**Hook line:** "Google just launched Pomelli. It's their new AI agent for marketing..."
**Source:** @nick_saraev | 38K likes / 648K plays
**Angle type:** Tool-Focus
**ICP:** Primary (TOFU)
**Hook pattern:** Product Drop
**Emotional register:** Excited / Empowered

**Why it works:**
- "Just launched" creates urgency and FOMO — viewer needs to know about this now
- Names the company (authority) + names the product (specific) + states the category ("AI agent for marketing") in 11 words — high information density, zero fluff

**Body structure:** What it is → what it does → the one capability worth knowing about
**CTA pattern:** Comment-gated → access or template
**Visual style:** Tool logo / product UI screenshot in split-screen

**Adapt like this:**
"[Company] just [launched/dropped/released] [Product Name]. It's [their/the] new [specific category]..."
Examples:
- "Anthropic just dropped Claude 3.7. It's their new reasoning model for code..."
- "n8n just launched AI Agents. It's their new native agentic workflow builder..."

### Example 4: Elimination (Free Alternative)

**Hook line:** "You don't need to pay for Veo 3 or Sora 2 anymore. Instead, just use this free open-source alternative..."
**Source:** @nick_saraev | 33K likes / 413K plays
**Angle type:** Concept
**ICP:** Primary (TOFU)
**Hook pattern:** Elimination
**Emotional register:** Challenged / Validated

**Why it works:**
- Names the expensive incumbents (Veo 3, Sora 2) — viewers who know these feel instant recognition and curiosity
- "You don't need to pay" is a direct permission slip — removes the objection before it forms
- "Free open-source" hits two top-performer keywords in 3 words

**Body structure:** Introduce the alternative → show it working → compare the result
**CTA pattern:** Comment-gated → setup guide or template
**Visual style:** Side-by-side comparison or tool logo reveal

**Adapt like this:**
"You don't need to [pay for / use / buy] [Named Expensive Tool] anymore. Instead, just use [free/open-source/built-in alternative]..."
Examples:
- "You don't need to pay for Zapier anymore. Instead, just use this free n8n setup..."
- "You don't need to hire a dev for this. Instead, just use Claude Code..."

**ICP:** Primary (TOFU)

**CTA pattern:** Comment-gated → breakdown or summary resource

### Example 6: Step-by-Step Tease (Design → Code Build)

**Hook line:** "How to build apps inspired by your favorite designs in 5 minutes..."
**Source:** @adrien.ninet | 24K likes / 434K plays
**Angle type:** Story
**ICP:** Primary (TOFU)
**Hook pattern:** Step-by-Step Tease
**Emotional register:** Curious / Energised

**Why it works:**
- "Inspired by your favorite designs" is aspirational and personal — viewer mentally fills in their own design reference
- "5 minutes" creates a clear, achievable promise — low perceived effort, high perceived reward
- Works without naming a specific tool in the hook — the curiosity gap does the work

**Body structure:** Pick a reference design → show the AI interpretation step → reveal the working app
**CTA pattern:** Comment-gated → template or workflow
**Visual style:** Design reference in top half → transition to code output / working app

**Adapt like this:**
"How to [build/create/ship] [desirable output] [inspired by / from / using] [relatable input] in [short time]..."
Examples:
- "How to build a client dashboard from a Figma screenshot in 10 minutes..."
- "How to ship an AI agent using nothing but a voice note in 5 minutes..."
