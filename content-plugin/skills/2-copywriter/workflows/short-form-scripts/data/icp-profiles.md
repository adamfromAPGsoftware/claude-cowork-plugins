# ICP Profiles — Short-Form Scripts

Workflow-scoped ICP reference. Load this file at script-write time to drive language register, hook pattern selection, and CTA keyword choice.

---

## Primary ICP — Aspiring AI Agency Builders (TOFU)

**Demographics:** Age 20–35, freelancers / junior devs / bootcamp grads
**Pain:** Gap between using ChatGPT casually and building production AI systems for clients. Technically curious but need a trusted guide who's already shipped.
**Goal:** Land first AI client, build a recurring-revenue agency, ship real automations.

**Language markers:** "build," "ship," "deploy," "Next.js," "Supabase," "n8n," "Claude Code," "Cursor," "agentic," "MCP," "workflow," "automation," "template"
**Emotional register:** Curious / Energised / Slightly impatient
**Hook priority:** 1st Step-by-Step Tease, 2nd Product Drop, 3rd Value-Drop
**MG visual style:** Coding interface, terminal output, workflow diagram, fast cursor movement, tool UI walkthrough
**CTA keywords (TOFU):** AGENT, BUILD, CLAUDE, CURSOR, TEMPLATE, WORKFLOW, AUTOMATE, STACK, n8n, DEPLOY, SHIP

---

## Secondary ICP — SME Decision Makers (BOFU)

**Demographics:** Age 30–55, SME owners / CTOs / Operations leads, 5–100 employee companies
**Pain:** $12K–$24K/year wasted on disconnected SaaS subscriptions; 10–25 hrs/week lost to manual, repetitive work that no one has time to fix.
**Goal:** Reduce tool sprawl and labour cost. Proof before they invest.
**Trigger:** Has tried to automate before and failed, or just saw a competitor do something smarter.

**Language markers:** "ROI," "efficiency," "unified system," "SaaS waste," "cost savings," "hours saved," "headcount," "operations," "scale," "revenue impact"
**Emotional register:** Pragmatic / Cost-sensitive / Proof-hungry
**Hook priority:** 1st Result Reveal, 2nd Pain Interrupt, 3rd Before/After
**MG visual style:** Cost counter, side-by-side comparison, before/after dashboard, SaaS logo wall with X marks, time-savings graphic
**CTA keywords (BOFU — use ONLY these):** AUDIT, AI, RESULTS, ROI, SYSTEM

## 5-Angle / ICP Matrix

| Angle Type | ICP Target | Hook Patterns | Emotional Register | MG Visual Default |
|---|---|---|---|---|
| Story | Primary (TOFU) | Journey Tease, Step-by-Step Tease | Curious / Energised | Coding interface, workflow diagram |
| Tool-Focus | Primary (TOFU) | Product Drop, Result Reveal | Excited / Empowered | UI walkthrough, tool logo |
| Concept | Primary (TOFU) | Pain Interrupt, Elimination | Challenged / Validated | Abstract tech, counter graphic |
| Value-Drop | Primary (TOFU) | Value-Drop, Step-by-Step Tease | Relieved / Grateful | Clean logo badge, high-level workflow |
| Before/After | Secondary (BOFU) | Result Reveal, Pain Interrupt | Pragmatic / Proof-hungry | Cost counter, side-by-side comparison |

### Batch Diversity Rule

- **Minimum 1 Secondary ICP (Before/After)** script per batch of 5
- **Maximum 2 scripts of any single angle type** in a batch
- **Minimum 2 distinct emotional registers** across the 5 scripts

## Language Register Guide

Same core insight, reframed per ICP. Use these as translation anchors when writing each script.

| Insight | Primary (TOFU) — Builder Voice | Secondary (BOFU) — Decision Maker Voice |
|---|---|---|
| Automation | "Deploy an n8n workflow that handles this automatically — here's the exact setup." | "Eliminate 8 hours per week per team member. No new hires needed." |
| Cost | "This is completely free to get started — open source, self-hosted." | "Replaces $2,400/month in SaaS subscriptions — same outcome, one system." |
| Speed | "I shipped this in 2 hours with Claude Code and Cursor." | "Implementation is done in under a week. ROI inside the first month." |
| Complexity | "You just wire up the MCP server, point it at your codebase, and it handles the rest." | "Your team doesn't need to touch the tech — we handle setup, you get the results." |
| CTA | "Comment BUILD and I'll send you the template." | "Comment AUDIT and I'll send you the ROI checklist." |

**Rule:** Before writing a Secondary ICP script, run every body sentence through this test — "Would a 45-year-old SME owner care about this?" If the sentence is about syntax, tooling choices, or developer workflow internals, reframe it as a business outcome.
