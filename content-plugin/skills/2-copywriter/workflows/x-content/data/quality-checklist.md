# X/Twitter Quality Checklist

Format-specific quality gates for X content review.

---

## Universal Checks (All Formats)

- [ ] **Hook lands in ≤140 chars** — visible above the "Show more" fold on mobile
- [ ] **No forced engagement bait** — "like and RT if you agree" style CTAs absent
- [ ] **Hashtags ≤1** — zero preferred; only if a highly relevant trending tag adds value
- [ ] **No LinkedIn phrasing** — "excited to share", "humbled by", "lessons learned" absent
- [ ] **Voice is confident, not hedged** — "I think maybe" → "This is"
- [ ] **ICP relevant** — content targets at least one known ICP segment
- [ ] **Not a duplicate angle** — derivative tracking confirms no hook/angle duplication
- [ ] **One CTA maximum** — multiple CTAs removed or consolidated

---

## Single Post (≤280 chars)

- [ ] **Character count ≤280** — count displayed; URLs counted as 23 chars
- [ ] **Hook is standalone** — first sentence works as the entire post if read alone
- [ ] **No filler words** — every word earns its place
- [ ] **Specificity check** — at least one number, outcome, or concrete detail
- [ ] **CTA is natural** — not bolted on; fits the tone of the post

---

## Thread (2–20 tweets)

- [ ] **Tweet count in sweet spot** — 5–12 tweets; flag if under 5 or over 12
- [ ] **Tweet 1 stands alone** — hook tweet functions as a complete single post
- [ ] **Thread signal present** — 🧵 or "Here's how:" or "Thread:" in tweet 1
- [ ] **One idea per tweet** — no tweet cramming two distinct points
- [ ] **Numbered if listicle** — consistent numbering (1., 2., 3.) throughout body tweets
- [ ] **Bookmark hook placed** — appears in tweet 1 or 2 (not just the final tweet)
- [ ] **Final tweet has CTA** — follow hook + engagement/RT prompt
- [ ] **Final tweet has summary** — 1-2 line thread summary before the CTA
- [ ] **Each tweet is ≤280 chars** — count all tweets individually
- [ ] **No tweet is orphaned** — each tweet connects logically to the next

---

## Long Post (≤25,000 chars)

- [ ] **Opening paragraph is gripping** — functions as hook standalone
- [ ] **Character count reasonable** — flag if over 5,000 chars without justification
- [ ] **Structure is skimmable** — headers or bold text section markers present
- [ ] **Clear argument arc** — problem → insight → implication/solution
- [ ] **Closer is strong** — final section delivers clear takeaway, not fade-out
- [ ] **Follow CTA at end** — earns the follow after delivering value
- [ ] **No padding** — long ≠ better; every section adds value

---

## Image Post

- [ ] **Caption amplifies, doesn't duplicate** — caption adds POV the image doesn't show
- [ ] **Caption fits above fold** — ≤140 chars preferred (may extend with justification)
- [ ] **Image is self-explanatory** — viewer understands the image without needing the caption
- [ ] **No text-heavy image** — minimal text in the image itself (reduces reach)
- [ ] **File within specs** — JPEG/PNG ≤5MB, or GIF ≤15MB

---

## Video Post

- [ ] **First line = visual promise** — viewer knows exactly what they're about to see
- [ ] **Caption is brief** — ≤3 lines
- [ ] **First 3 seconds work silent** — autoplay is muted; opening frame is compelling
- [ ] **Video delivers on caption promise** — no bait-and-switch
- [ ] **File within specs** — MP4/MOV; standard ≤2:20 min and ≤512MB

---

## Content Category Checks

### Lead Magnet Posts
- [ ] **Reply keyword is clear** — one word, specific, relevant to content
- [ ] **Resource is real** — the deliverable being offered exists
- [ ] **CTA is prominent** — keyword trigger is clearly communicated

### Personal Posts
- [ ] **Story is specific** — concrete details, real moment, not vague reflection
- [ ] **Authentic voice** — reads like a person, not a content producer
- [ ] **CTA invites dialogue** — conversation starter, not broadcast

### Nurture / Educational Posts
- [ ] **Value is genuine** — actionable insight, not surface-level advice
- [ ] **Expertise visible** — content demonstrates knowledge, not just opinions
- [ ] **CTA is soft** — bookmark, follow, or share — no hard sell
