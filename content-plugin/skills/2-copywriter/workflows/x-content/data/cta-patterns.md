# X/Twitter CTA Patterns

CTA archetypes, usage guidance, and placement rules for X content.

---

## CTA Philosophy on X

X CTAs differ significantly from LinkedIn:
- **No keyword DM funnels** as default — X readers are more sceptical of "Comment KEYWORD"
- **Engagement > conversion** as primary CTA for most posts
- **RT triggers** can significantly expand reach
- **Bookmark hooks** are underused and highly effective for threads
- **Follow CTAs** belong at the end of threads, not single posts

Key rule: one CTA per post. Multiple CTAs dilute attention and read as desperation.

---

## CTA Types by Format

### Single Post CTAs
Single posts have tight character budgets. CTAs must be:
- 1 line maximum
- Integrated naturally into the post voice
- Not the dominant feature of the post

**Best for single posts:**
- Reply bait: "What would you add?"
- Hot take challenge: "Agree or disagree?"
- Soft share: "Know someone who needs this?"

---

### Thread CTAs
Threads have dedicated final tweet real estate for CTAs. Use it.

**Final tweet CTA structure:**
```
[1-line thread summary or punchline]

Follow @[handle] for more [topic].

[RT hook or bookmark hook]
```

**Mid-thread bookmark hooks (tweet 1 or 2):**
```
Save this thread. You'll need it.
```
```
Bookmark this. Come back when you need it.
```

---

### Long Post CTAs
End of long post — reader has invested time, so the CTA ask can be slightly stronger.

```
If this helped, follow @[handle] for more [topic].

[Share / RT prompt if appropriate]
```

---

### Image/Video Post CTAs
Keep short. Image and video are the focus.

```
[1-line question or reaction prompt]
```
```
Follow for more like this.
```

## CTA Pattern Library

### 1. Reply Bait
Invites replies — boosts engagement signals to the algorithm.

```
What would you add?
```
```
Drop your take below.
```
```
What am I missing?
```
```
Hot take or obvious? Tell me.
```
```
Reply with [X] if you've experienced this.
```

**Best for:** Single posts, threads (final tweet), personal content, nurture posts
**Avoid:** Lead magnet posts where you need a specific reply trigger

---

### 2. RT / Share Triggers
Expands reach. Works best when the post has broad appeal.

```
RT if this changed how you think about [topic].
```
```
Share this with someone who needs it ↑
```
```
The more people who know this, the better. RT.
```
```
Tag someone building [thing] who needs to see this.
```

**Best for:** Educational/nurture threads, hot takes with strong POV
**Avoid:** Personal stories (RT asks feel out of place)

---

### 3. Follow Hooks
Builds audience. Use at the END of threads — earns the follow.

```
Follow @[handle] for more [topic] breakdowns.
```
```
If this thread helped — follow. More like this every week.
```
```
I write about [topic] here. Follow if that's useful.
```

**Best for:** Final tweet of threads, end of long posts
**Never:** Mid-post or on single posts (feels desperate)

---

### 4. Bookmark Hooks
Converts passive scrollers into engaged readers who return.

```
Save this thread. You'll need it.
```
```
Bookmark this. Come back when you're ready to use it.
```
```
This is the kind of thing you'll want to reference later.
```

**Best for:** Thread tweet 1 or 2, how-to threads, reference-heavy content
**Placement:** Tweet 1 or 2 in a thread (not last — bookmark before they scroll away)

---

### 5. Lead Magnet / Resource CTA
Reply-to-receive DM funnels. Less common on X than LinkedIn but still used.

```
Reply [KEYWORD] and I'll send you [resource].
```
```
DM me "[KEYWORD]" for the full [guide/template].
```

**When to use:** Only when there's a genuinely valuable resource to offer; overuse damages credibility
**Best for:** Lead magnet category posts where list-building is the goal
**Important:** The keyword should be highly relevant to the content — not generic

---

### 6. Conversation Starters
For personal/opinion posts. Invites genuine dialogue.

```
What's your experience with this?
```
```
Agree? Or am I wrong here?
```
```
Would love to hear the counterargument.
```
```
What would you have done differently?
```

**Best for:** Personal category posts, hot takes, story posts
**Tone:** Genuine curiosity, not performative engagement-seeking

---

## CTA Selection by Content Category

| Category | Primary CTA Style | Secondary |
|---|---|---|
| Lead Magnet | Reply [KEYWORD] / DM trigger | Bookmark hook |
| Personal | Conversation starter / reply bait | Soft RT trigger |
| Nurture / Educational | Bookmark hook / follow hook | RT trigger |

---

## Anti-Patterns (Never Use)

- "Like + RT if you agree" — feels like 2015 Twitter; X readers hate this
- "Follow me for daily tips" — generic; earn the follow, don't demand it
- Multiple CTAs in one post — dilutes focus
- CTA before delivering value — earn the ask first
- "I hope this helps!" before a CTA — weak framing
